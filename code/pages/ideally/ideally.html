Ideally a website should ...

To complete this sentence is almost impossible.

We all know designing and developing for the web is a herculean 1 task. Even if there are no more problems on the front-end 2 no one can create the perfect website.

So why this post? At least we can try. Many of us are thinking / dreaming / working on a perfect website. Even if the success is ephemeral — since web technology is ephemeral —, at least we can taste it for a second.

Massimo Vignelli was lucky. He had Photoshop, fixed viewport, no webservers, no SEO, no browsers, no bugs. As disciples we should follow him, no matter the context and the medium became chaotic, universally brutal.
